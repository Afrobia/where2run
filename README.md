## Project

### Run project locally:
`npm run dev`<br/>
http://localhost:3000

### Build projetc:
`npm rum build`

### Start build project:
`npm start`

### Run tests:
`npm test`<br/>
`npm run test:watch`

### Run eslint
`eslint src`

### Fix files
`eslint src --fix`

`npm instal --include=dev @types/leaflet`
